# Backend 'Usuario'

#### **[+] 'End Points'**
Todas los usuarios:
>**method: get**\
>/users

Un usuario en especifico:
>**method: get**\
>/user/:id

Agregar un usuario:
>**method: post**\
>/user

Actualizar un usuario:
>**method: put**\
>/user/:id

Eliminar un usuario:
>**method: delete**\
>/user/:id
